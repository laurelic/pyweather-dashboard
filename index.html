Background

The aim of this project is to observe weather patterns depending on a location's proximity to the equator. Though we might assume that the temperature increases, we also know that weather involves more than temperature. Not to mention, it is healthy to confirm our assumptions before proceeding. As such, we utilized the OpenWeatherMap API to survey 500 randomly chosen cities for temperature, cloud cover, wind speed, and humidity. These factors were visualized according to cities' latitudinal position.

This site presents those visuals along with source code and analyses observed.
